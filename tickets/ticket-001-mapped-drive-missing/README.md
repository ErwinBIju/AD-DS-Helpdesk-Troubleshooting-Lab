## Initial Hypothesis

The mapped drive may be missing because of one or more of the following:

- The GPO is not linked to the correct OU.
- The user is not in the correct OU.
- The mapped drive policy is misconfigured.
- Security filtering is preventing the GPO from applying.
- The user does not have permission to access the shared folder.
- The user has not logged off and back on after a group membership change.
