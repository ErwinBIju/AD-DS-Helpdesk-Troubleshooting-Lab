
# Network Diagram

This folder contains the network diagram for the AD DS troubleshooting lab.

## Planned Lab Topology

- DC01 - Windows Server 2022 Domain Controller
- Windows 11 domain-joined client
- Domain: lab.local
- Network: 192.168.40.0/24
- DNS handled by DC01
- VMware Workstation lab network

## Purpose

The diagram will show how the domain controller, client machine, DNS, and lab network are connected.
