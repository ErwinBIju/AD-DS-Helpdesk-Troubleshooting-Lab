# Lab Environment

## Purpose

This lab simulates a small business Active Directory environment used to practice help desk and junior system administrator troubleshooting.

## Infrastructure

| Component         |              Details                      |
|-------------------|-------------------------------------------|
| Domain            | lab.local                                 |
| Domain Controller | DC01                                      |
| Server OS         | Windows Server 2022                       |
| Client OS         | Windows 11 Enterprise                     |
| Network           | 192.168.40.0/24                           |
| Virtualization    | VMware Workstation                        |
| DNS Server        | DC01                                      |
| DHCP              | DC01 or lab router/NAT depending on ticket|

## Core Services Practiced

- Active Directory Domain Services
- DNS
- Group Policy
- File sharing
- NTFS and share permissions
- Kerberos authentication checks
- Windows client troubleshooting

## Documentation Method

Each ticket includes:

- Issue summary
- Symptoms
- Initial hypothesis
- Troubleshooting steps
- Commands used
- Root cause
- Fix
- Verification
- Interview explanation
